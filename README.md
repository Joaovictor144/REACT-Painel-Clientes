## 👍🏻 Como baixar

```
    // Clonar repositório
    git clone git@github.com:Joaovictor144/REACT-Painel-Clientes.git
    // Acessar diretório
    cd REACT-Painel-Clientes
     // Instalar dependências
     yarn
    // Iniciar projeto
    yarn start
```
